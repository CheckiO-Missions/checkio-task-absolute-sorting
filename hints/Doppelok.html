# For example, sort lists by sums.
c = sorted([[90, 11], [99, 1], [200], [30, 50, 50]], key=sum)

Look, look! In general, you don't need to use sorted function. If you have deal with lists use list method list.sort(). But careful, this method change the list, but return None.

    values.sort(key = lambda x: abs(x))
