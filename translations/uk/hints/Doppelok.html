Добре, добре! Оце настрій!=) Це рішення використовує lambda-функцію. Бачиш, що наше попереднє рішення можна записати в однорядковий вираз. 

checkio = lambda n: sorted(n, key=abs)

Я хочу більше!

Подивись на цей розв'язок. Ти можеш поєднати функцію sorted та lambda інакше. 

def checkio(numbers_array: list) -> list:

    return sorted(numbers_array, key=lambda x: abs(x))

Дивись, дивись! Взагалі, ти можеш не використовувати функцію sorted. Якщо ти маєш справу зі списком, можеш використати метод списків list.sort(). Але обережно, цей метод змінює сам список, але повертає у відповідь None. 

def checkio(values: list) -> list:

    values.sort(key=lambda x:abs(x))

    return values
