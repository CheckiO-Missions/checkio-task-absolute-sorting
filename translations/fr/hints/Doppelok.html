def checkio(numbers_array: tuple) -> list:

    return sorted(numbers_array, key=lambda x: abs(x))

Regardez, regardez! En général, vous n'avez pas besoin d'utiliser la fonction de tri. Si vous avez affaire à des listes, utilisez la méthode list list.sort(). Mais attention, cette méthode modifie la liste, mais renvoie None. 

def checkio(values: list) -> list:

    values.sort(key = lambda x: abs(x))

    return values
