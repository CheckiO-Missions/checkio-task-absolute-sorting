您应该尝试使用 the key parameter for sorted function. 

# For example, sort lists by sums.
c = sorted([[90, 11], [99, 1], [200], [30, 50, 50]], key=sum)
# Function "sum" in "key" will be executed for each element to decide the order.
print(c)

输出：

[[99, 1], [90, 11], [30, 50, 50], [200]]

key 不改变元素。

听着，听着！一般来说，你不需要使用sorted函数。如果你要处理列表，使用列表方法list.sort() 。但是要小心，这个方法改变了列表，但是返回None 。 

def checkio(values: list) -> list:

    values.sort(key = lambda x: abs(x))
